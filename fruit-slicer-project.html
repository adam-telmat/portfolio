Fruit Slicer Game

A Python recreation of the popular Fruit Ninja mobile game

Project Overview

Fruit Slicer is my recreation of the popular mobile game Fruit Ninja, built entirely in Python. The game challenges players to slice fruits that appear on screen while avoiding bombs. This project demonstrates my skills in game development, physics simulation, and creating engaging user experiences.

I developed this game to deepen my understanding of Python's capabilities for game development and to practice implementing real-time physics and collision detection algorithms. The project showcases my ability to create an interactive application with smooth animations and responsive controls.

Technologies Used

Python Pygame Object-Oriented Programming Physics Simulation Collision Detection

Key Features

  • Real-time fruit slicing mechanics with realistic physics
  • Score tracking and combo system
  • Multiple fruit types with different behaviors
  • Bombs and special items to avoid or collect
  • Particle effects for visual feedback
  • Increasing difficulty as the game progresses
